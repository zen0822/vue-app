# vue 单页应用和多页应用的模板

## mpa 多页应用模板

### 构建开发环境

```sh
node bin/dev --app mpa

# 或者运行 npm 脚本
# npm run devMpa
```

### 构建测试环境

#### 单元测试

```sh
node bin/test --app mpa

# 或者运行 npm 脚本
# npm run testMpa
```

### 打包产品环境的代码

```sh
node bin/prod --app mpa

# 或者运行 npm 脚本
# npm run prodMpa
```
